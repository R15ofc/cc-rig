# Dock

Dock is the OS layer for CC:Tweaked computers.

## Editions

- **Dock Pocket** - Pocket Computer UI for apps, account, app store, notifications, and basic tools.
- **Dock PC** - Advanced Computer UI for developer tools, terminals, package management, and richer apps.

## Product Goals

- app launcher;
- account login;
- app store;
- settings;
- notifications;
- package install/remove/update UX;
- app permissions;
- developer mode on advanced computers.
