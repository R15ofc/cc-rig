# Platform Roadmap

## Phase 1 - RIG Dev Foundation

- Keep current hub running.
- Convert RIG CLI from monitoring-first to dev-tool-first.
- Add package upload flow from GitHub/gist.
- Add package trust states.
- Add package warning/remove/block behavior.
- Add package source metadata and permissions.

## Phase 2 - Dock OS

- Build Dock Pocket Computer shell.
- Build Dock Advanced Computer shell.
- Add app launcher, settings, account login, app store, notifications.
- Add package install/update/remove UI.
- Add app permissions prompts.

## Phase 3 - Luma Internet

- Build Luma browser app.
- Add domain/directory registry in hub.
- Add search index.
- Add basic site format.
- Add site builder templates.
- Add publish-to-Luma flow through RIG packages.

## Phase 4 - Developer Ecosystem

- SDK docs.
- App templates.
- CLI scaffolding commands.
- Package signing/verification metadata.
- Hub moderation tools.
- App store review queue.
